FlipStar

Инструкция и команды для плагинов:

Установка:

Закинуть содержимое папки "Сборка" в корень игры.

Команды:

+ /rec - реконнект
+ /fconnect 1 50 - Fixconnect by AIR
+ /fd - туман /dd - прорисовка(добавляеть FPS) /ld - лоды
+ /st - время менять /sw - погоду менять. ( /bt /bw)
+ Cбив anim на X (Ч) "могут дать варн!"
+ /mt - Multi Tools by RCM ( узнать все функции Тык) Ps /sskin есть тут
